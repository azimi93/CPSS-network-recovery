## 📖 Overview

This repository contains the implementation of the paper: Restoring Structural Controllability of Interdependent Multi-layered Heterogenous Cyber-Physical-Social Networks via Optimal Network Reconfiguration. The framework addresses the challenge of restoring controllability when a network becomes uncontrollable due to node or edge failures, attacks, or cascading disruptions across interdependent layers.

### Key Contributions

| Contribution | Description |
|:-------------|:------------|
| **Structural Controllability Restoration** | Identifies minimal network reconfiguration actions to restore controllability |
| **Multi-layer Interdependency Modeling** | Captures complex couplings between cyber, physical, and social networks |
| **Optimal Reconfiguration** | Solves for minimum-cost edge additions or node activations |
| **Heterogeneous Dynamics** | Handles different node dynamics across layers (e.g., power, communication, social) |
